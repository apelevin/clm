# Contract Parser UI

Веб-приложение для парсинга договоров с использованием OpenAI API. Извлекает структурированную информацию из текста договора, выделяет действия сторон и ключевые положения, а также позволяет отслеживать связь между интерфейсными элементами и исходным текстом.

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env.local` в корне проекта:
```
OPENAI_API_KEY=your_openai_api_key_here
```

3. Запустите dev-сервер:
```bash
npm run dev
```

4. Откройте браузер по адресу [http://localhost:3000](http://localhost:3000)

## Использование

1. Вставьте текст договора в текстовое поле или загрузите `.txt` файл
2. Нажмите "Обработать договор"
3. Дождитесь завершения обработки
4. Просмотрите результат:
   - Левая колонка: исходный текст договора
   - Правая колонка: карточка договора, действия и ключевые положения
5. Нажмите "Показать источник" на любой карточке, чтобы подсветить соответствующие параграфы в тексте

## Структура проекта

- `app/` - Next.js App Router страницы и API routes
- `components/` - React компоненты
- `lib/` - Утилиты и клиенты (OpenAI, парсеры)
- `types/` - TypeScript типы

## Технологии

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- OpenAI API (GPT-4 Turbo)
